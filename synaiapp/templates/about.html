What do you mean by audio features ?

Audio features are a set of values calculated by Spotify on a defined song.
Let's take the danceability value as an example.
Danceability describes how suitable a track is for dancing based on a combination of musical elements including tempo, rhythm stability, beat strength, and overall regularity. A value of 0.0 is least danceable and 1.0 is most danceable.

How do the recommendations work ?

The Spotify API provides us a nice way to recommend songs. For a playlist for example, we are going to take 5 songs as a sample. We also use the audio features we analysed to make the songs recommended more precise. We do not know exactly how the algorithm behind works as this is kept secret by Spotify.
